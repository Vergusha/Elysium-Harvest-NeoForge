# Texture Instructions for Florite Set Bonus Effect

## Location
Place the effect icon texture at:
`src/main/resources/assets/elysiumharvest/textures/mob_effect/florite_set_bonus.png`

## Specifications
- **Size**: 18x18 pixels (standard Minecraft effect icon size)
- **Format**: PNG with transparency
- **Style**: Should match Minecraft's effect icon style

## Design Suggestions
Since this is a crop growth acceleration effect from Florite armor, the icon could feature:
1. A green/emerald colored plant or wheat symbol (representing crop growth)
2. Sparkles or particles around it (representing the magical effect)
3. A florite-themed border or accent (to tie it to the armor set)
4. Green tones (#4CAF50) to match the effect color defined in code

## Reference
Look at vanilla Minecraft effect icons for style guidance:
- Growth/Nature effects like Regeneration
- Beneficial effects with plant/nature themes

## Temporary Solution
If you don't have a texture yet, the effect will still work but will show as a purple/black checkerboard pattern in the inventory screen.
