# Qazan GUI Texture

## Требования к текстуре qazan.png

Необходимо создать PNG файл `qazan.png` размером **256x256 пикселей**.

### Структура текстуры:

#### Основная GUI (0, 0, 176, 166):
- Фон контейнера стандартного размера
- **Слоты для ингредиентов**: 6 слотов в 2 ряда по 3
  - Позиции: (44, 17), (62, 17), (80, 17) - первый ряд
  - Позиции: (44, 35), (62, 35), (80, 35) - второй ряд
  
- **Слот для результата**: 1 слот справа
  - Позиция: (116, 26)

- **Прогресс-бар**: область между ингредиентами и результатом
  - Позиция: (103, 26)
  - Размер: 24x16 пикселей

- **Инвентарь игрока**: стандартная область внизу
  - 3 ряда по 9 слотов
  - Хотбар: 9 слотов

#### Текстура прогресс-бара (176, 0, 24, 16):
- Анимация огня/приготовления
- Заполняется слева направо при готовке

### Цветовая схема:
- Используйте стандартную майнкрафтовскую палитру для GUI
- Фон: темно-серый (#373737)
- Обводка слотов: более светлый серый (#8B8B8B)
- Прогресс-бар: огненные тона (оранжевый/красный)

### Альтернатива:
Вы можете взять за основу текстуру печи (furnace.png) из стандартного ресурспака Minecraft и адаптировать её под свои нужды.
