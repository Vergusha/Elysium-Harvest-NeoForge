ИНСТРУКЦИЯ: Замена текстур ванильных мотыг
==========================================

Поместите сюда свои PNG файлы (16x16 пикселей) со следующими именами:

1. wooden_hoe.png    - Деревянная мотыга
2. stone_hoe.png     - Каменная мотыга
3. iron_hoe.png      - Железная мотыга
4. golden_hoe.png    - Золотая мотыга
5. diamond_hoe.png   - Алмазная мотыга
6. netherite_hoe.png - Незеритовая мотыга

После добавления файлов, этот README можно удалить.

Требования:
- Формат: PNG
- Размер: 16x16 пикселей (стандартный размер предметов Minecraft)
- Прозрачный фон (альфа-канал)

Эти текстуры заменят ванильные мотыги для всех, кто установит ваш мод.
