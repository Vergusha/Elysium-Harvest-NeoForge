# Qazan Block - Инструкция по финализации

## Проблемы с API в Minecraft 1.21.10

В версии 1.21.10 Minecraft + NeoForge произошли значительные изменения в API. Код требует следующих исправлений:

### 1. Проверьте правильный метод доступа к `isClientSide`
В новой версии возможно, что `level.isClientSide` изменился на `level.isClientSide()` - метод вместо поля.

### 2. Обновите методы сохранения/загрузки NBT
API для BaseContainerBlockEntity изменился. Возможно требуется использовать другие методы для NBT операций.

### 3. Исправьте метод blit() в QazanScreen
Метод отрисовки текстур изменился в GuiGraphics.

### 4. Создайте текстуру GUI
Необходимо создать PNG файл по пути:
`src/main/resources/assets/elysiumharvest/textures/gui/qazan.png`

См. README_QAZAN_TEXTURE.md для спецификации.

## Функционал Qazan:

- 6 слотов для ингредиентов
- 1 слот для результата
- Работает только при наличии источника тепла снизу (печь, костёр, магма, лава)
- Правый клик с миской в руке забирает готовую еду
- Правый клик без миски открывает GUI
- Прогресс-бар показывает процесс готовки (10 секунд = 200 тиков)
